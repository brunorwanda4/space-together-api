## Space-Together API - School Management System

This is a comprehensive school management backend built with Rust (Actix-web framework) and MongoDB. It's designed as a multi-tenant SaaS platform for educational institutions.

### Core Architecture

- **Multi-tenant system**: Each school gets its own isolated database, identified via subdomain or X-School-ID header
- **RESTful API**: Built with Actix-web, running on port 4646
- **MongoDB**: Uses separate databases per school for data isolation
- **JWT Authentication**: Token-based auth with role-based access control
- **Dockerized**: Ready for containerized deployment

### Key Features

**User Management**
- Multiple user roles: Admin, Staff, Student, Teacher
- Rich user profiles with personal info, contact details, social media
- Guardian information for students
- Teaching credentials and availability for teachers
- Profile customization with images and backgrounds

**School Administration**
- School registration and management
- Support for different school types (Public, Private, Charter, International)
- School metadata: accreditation, affiliation, capacity, facilities
- Multi-school support (users can belong to multiple schools)

**Academic Structure**
- Sectors and trades (curriculum/education levels)
- Classes and subclasses with hierarchical organization
- Class types and levels
- Template subjects for standardized curriculum
- Class subjects with teacher assignments
- Education year management

**Timetable Management**
- School-wide timetables
- Class-specific timetables
- Period types and day structure configuration
- Flexible scheduling system

**Student Management**
- Student enrollment and registration
- Student status tracking (Active, Suspended, Graduated, Left)
- Class assignments
- Learning challenges and special support needs
- Study preferences and favorite subjects

**Teacher Management**
- Teacher types: Regular, HeadTeacher, SubjectTeacher, Deputy
- Subject and class assignments
- Teaching credentials and certifications
- Availability scheduling

**Staff Management**
- School staff profiles
- Department and job title tracking
- Employment type management

**Communication & Social**
- Announcements system
- Comments and likes functionality
- Join school requests
- Event management
- Notification system (infrastructure present)
- Chat service (infrastructure present)

**Attendance & Finance**
- Attendance tracking system
- Finance management (modules present)
- Support for different attendance systems (Manual, Online)

**Additional Features**
- Database status monitoring
- Cloudinary integration for image uploads
- Event bus for internal messaging
- Aggregation pipelines for complex queries
- Comprehensive filtering and pagination
- Role-based guards for endpoint protection
- CORS enabled for cross-origin requests

### Technical Highlights

- **Performance**: Optimized with connection pooling and async operations
- **Security**: JWT tokens, password hashing with bcrypt/argon2, role guards
- **Scalability**: Multi-tenant architecture with database per school
- **Flexibility**: Extensive configuration options for different school types
- **Data Integrity**: MongoDB with structured schemas and validation

The application serves as a complete backend for educational management platforms, handling everything from user authentication to academic scheduling, with a focus on multi-school support and data isolation.
